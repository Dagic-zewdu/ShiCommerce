# ShiShop

> ShiShop is an e-comerce website built with NodeJS, ExpressJS, MongoDB, and ReactJS.

## Built With

- NodeJS
- ExpressJS
- MOndoDB
- ReactJS
- Redux
  )

## Getting Started

To get the copy of this project in your computer,clone this repository or just download it.

### Prerequisites

- Code editor like vs code
- node js installed

### Setup

- Download or clone the repository
- Open the folder in your IDE
- Install the dependencies
- Open the index.html in your browser and check it
- Add your addition
- Push to your github or deploy to cloud
